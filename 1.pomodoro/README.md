# ポモドーロタイマー - ゲーミフィケーション版

## 概要
このポモドーロタイマーアプリケーションは、モチベーション維持と継続利用を促進するゲーミフィケーション要素を備えた集中作業支援ツールです。

## 主な機能

### 🎮 ゲーミフィケーション要素

#### 1. 経験値(XP)システム
- ポモドーロ完了ごとに **100 XP** を獲得
- ストリークボーナス:
  - 3日連続: +50 XP
  - 7日連続: +100 XP
- レベルアップ: 必要XP = レベル × 200

#### 2. 達成バッジシステム
- 🎯 **はじめの一歩**: 初めてのポモドーロを完了
- 🔥 **3日連続**: 3日連続でポモドーロを完了
- 🔥🔥 **1週間連続**: 7日連続でポモドーロを完了
- ⭐ **週間マスター**: 1週間で10回のポモドーロを完了
- 🌟 **レベル5**: レベル5に到達
- 🌟🌟 **レベル10**: レベル10に到達

#### 3. ストリーク機能
- 連続達成日数を自動カウント
- ストリーク継続でボーナスXP獲得
- 最長ストリーク記録の保持

#### 4. 統計ダッシュボード
- **週間統計**: 完了セッション数、総集中時間、平均時間
- **日別グラフ**: 過去7日間の完了数を視覚化
- **履歴**: 最近20件のセッション履歴

## セットアップ

### 必要な環境
- Python 3.8以上
- pip

### インストール手順

1. 依存パッケージのインストール:
```bash
cd 1.pomodoro
pip install -r requirements.txt
```

2. アプリケーションの起動:
```bash
python app.py
```

3. ブラウザでアクセス:
```
http://localhost:5000
```

## 使い方

### 基本的な使用方法

1. **タイマーの開始**
   - 「スタート」ボタンをクリックしてタイマーを開始
   - タスク名を入力（任意）

2. **ポモドーロの完了**
   - タイマーが0:00になったら自動的に通知
   - または「完了」ボタンを手動でクリック
   - XPとバッジを獲得

3. **統計の確認**
   - 「バッジ」タブ: 獲得したバッジを確認
   - 「統計」タブ: 週間の進捗とグラフを確認
   - 「履歴」タブ: 過去のセッションを確認

### ゲーミフィケーション要素の活用

- **毎日継続**: ストリークを維持してボーナスXPを獲得
- **バッジ収集**: 様々な条件を達成してバッジをコンプリート
- **レベルアップ**: XPを貯めてレベルを上げ、達成感を味わう

## 技術スタック

- **バックエンド**: Flask (Python)
- **データベース**: SQLite (Flask-SQLAlchemy)
- **フロントエンド**: HTML, CSS, JavaScript (Vanilla)

## データベース構造

### User (ユーザー)
- XP、レベル、ストリーク情報を保存

### PomodoroSession (セッション)
- 完了したポモドーロの履歴を保存

### Badge (バッジ)
- 獲得したバッジ情報を保存

## テスト目的

このアプリケーションは、ゲーミフィケーション要素がユーザーのモチベーションとタイマー継続利用に与える影響を測定するために開発されました。

### 測定指標
- 連続使用日数（ストリーク）
- 週間/月間の完了セッション数
- レベル進行度
- バッジ獲得率
