# 🍅 ポモドーロタイマー

カスタマイズ性の高いポモドーロテクニックタイマーアプリケーションです。

## 機能

### 時間設定
- **作業時間**: 15分、25分、35分、45分から選択可能
- **休憩時間**: 5分、10分、15分から選択可能

### テーマ
- **ライトテーマ**: 明るく見やすいデザイン
- **ダークテーマ**: 目に優しい暗めのデザイン
- **フォーカステーマ**: 集中しやすいミニマルなデザイン

### サウンド設定
- **開始音**: タイマー開始時のサウンド
- **終了音**: タイマー終了時のサウンド
- **Tick音**: 毎秒のカウント音（オプション）

### その他の機能
- プログレスバー表示
- セッションカウンター（完了数を記録）
- 設定の自動保存（ローカルストレージ）
- ブラウザ通知（セッション完了時）
- レスポンシブデザイン（モバイル対応）

## セットアップ

### 必要条件
- Python 3.8以上
- pip

### インストール

1. 依存関係をインストール:
```bash
cd 1.pomodoro
pip install -r requirements.txt
```

2. アプリケーションを起動:
```bash
python app.py
```

3. ブラウザで以下のURLにアクセス:
```
http://localhost:5000
```

## 使い方

1. **設定ボタン**をクリックして設定パネルを開く
2. お好みの**作業時間**と**休憩時間**を選択
3. お好みの**テーマ**を選択
4. **サウンド設定**を調整
5. **開始ボタン**をクリックしてタイマーを開始
6. 必要に応じて**一時停止**や**リセット**を使用

## ポモドーロテクニックとは

ポモドーロテクニックは、時間管理術の一つです：

1. 25分間作業に集中
2. 5分間休憩
3. これを4回繰り返す
4. 4回目の後は15-30分の長い休憩

このアプリケーションでは、これらの時間を自由にカスタマイズできます。

## テスト目的

このアプリケーションは、ユーザーの好みに合わせた柔軟な設定を提供し、カスタマイズ性がユーザー継続率にどの程度影響するかを測定するために開発されました。

以下の設定がユーザーごとに保存され、継続的な利用を促進します：
- 作業時間と休憩時間の設定
- テーマの選択
- サウンド設定
- セッションカウント

将来的には、これらの設定データとセッション完了数を分析することで、カスタマイズ性の効果を測定できます。
