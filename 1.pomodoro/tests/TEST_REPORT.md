## JavaScript テストについて

JavaScript のタイマーエンジンの状態遷移テストは `tests/README_JAVASCRIPT_TESTS.md` に詳細な実装ガイドを用意しました。

Jest を使用したテストケース例:
- 初期状態のテスト
- start/pause/resume/reset の状態遷移
- タイマー完了時の動作
- モード切替 (作業/休憩)
- 残り時間と進捗の計算
- コールバック機能

実行方法:
```bash
# Python テスト
pytest tests/ -v --cov=pomodoro

# JavaScript テスト (Jest インストール後)
npm test:js
```

## テスト実行方法

### 全テストの実行
```bash
cd 1.pomodoro
pip install -r requirements.txt
pytest tests/ -v
```

### カバレッジレポート付き実行
```bash
pytest tests/ -v --cov=pomodoro --cov-report=html
```

HTMLカバレッジレポートは `htmlcov/index.html` で確認できます。

### 特定のテストファイルのみ実行
```bash
pytest tests/test_services.py -v
pytest tests/test_api.py -v
pytest tests/test_app.py -v
```
